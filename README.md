# POP-Market
Blockchain-based second-hand trading site
### 1. Overview
- 사용자의 중고거래와 가상화폐 충전내역을 블록으로 저장
- 합의 알고리즘에 따른 블록 생성으로 안전성 보증
- 가상화폐를 결제수단으로 선정하여, 거래 대금의 중개자 역할 참여 
- 네고를 통해 가격을 낮추는 사람을 없애기 위하여 사전입금 후 중고거래완료시 입금처리 : 판매자의 이익에 초점

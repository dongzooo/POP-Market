# POP-Market
Blockchain-based second-hand trading site
### 1. Overview
- 사용자의 중고거래, 가상화폐 충전내역의 블록화
- 합의 알고리즘에 따른 블록 생성과정의 안전성 보증
- 가상화폐를 결제수단으로 입/출금행위의 중개자 역할 참여 
- 네고를 통해 가격을 낮추는 사람을 막을 수 있는 판매자의 이익에 초점 
- 자동매매를 만들고 싶다면 위 메서드를 활용하면 가능
